==== Slicing and Filtering

[cols="30,70"]
|===
|Description|Command
| All
| `jq .[]`

| First
|	`jq '.[0]'`

| Range
| `jq '.[2:4]'`

| First 3
| `jq '.[:3]'`

| Last 2
| `jq '.[-2:]'`

| Before Last
| `jq '.[-2]'`

| Select array of int by value
| `jq 'map(select(. >= 2))'`

| Select array of objects by value
m| `jq '.[] &#124; select(.id == "second")'`

| Select by type
| `jq '.[] &#124; numbers'` ^*^

|===
